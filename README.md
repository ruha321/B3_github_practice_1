# リポジトリを**りぽ**とかわいく呼びたいプロジェクト

## 概要

リポジトリは英語でrepoと略されるのをご存じですか？  
日本語でも**りぽ**とかわいく呼ぶことでGitやリポジトリといった難しそうなものに親しみが湧くと思うんですよ
なのでリポジトリの**リポ**呼びを広めていきたい，そんなプロジェクトです

## 特徴

インストール要らずで使えるプロジェクトです  
日常会話や軽い文章でどしどし使っていきましょう

## 動作環境

日本国内，日本語コミュニティ

## インストール方法

皆さんの脳内に**りぽ**という単語をすり込めばインストール完了です

## 使い方

日本語での会話中や軽めの記事を書いているときに，リポジトリのことを**りぽ**と呼んでいきましょう!
